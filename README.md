# Brand-managment-system

Fullstack microservices application for managing brands and ingredients.  
Built with **Spring Boot**, **API Gateway** and **Angular**.

---

## Features

- Manage brands (create, update, delete, view)
- Manage ingredients assigned to brands
- Microservices architecture
- API Gateway for routing requests
- Angular frontend UI
- REST communication between services
